## OpsPilot V3.0 版本日志

**版本发布时间：** 2024年10月12日

### 功能新增

| 模块     | 新增功能                                                                                                                                                              |
| :------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 工作室   | 新增工作室模块，支持对 Bot 进行管理，可以对 Bot 进行上线、下线，回复规则与对话记录管理等操作。为了适用不同的场景，比如服务台知识库问答，运维专门问答等场景，可以设置不同的机器人。                                    |

| 供应商   | 新增供应商功能块，可以对 OpsPilot 能够使用的相关模型进行管理和配置。                                                                                                         |
