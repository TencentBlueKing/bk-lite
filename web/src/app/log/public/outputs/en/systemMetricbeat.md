**Basic Functions**
  - CPU usage
  - Memory usage
  - Disk metrics
  - File system monitoring
  - Network metrics
  - Process monitoring
  - System runtime status

  - Modular design
  - Lightweight & real-time
  - Cross-platform support
  - Visualization and integration
  - Suitable for various application scenarios
