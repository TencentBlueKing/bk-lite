# 2026年6月26日 更新日志
版本发布时间：2026年6月26日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| CMDB | 新增机房机柜视图，支持机柜布局与 U 位利用率展示 |
| CMDB | 新增 IP 地址管理（IPAM）能力，支持网段规划、IP 自动发现扫描与自动对账冲突管理 |
| CMDB | 扩充配置采集对象，新增华为存储、MinIO、InfluxDB 等采集能力 |
| 监控中心 | 新增 117 个 SNMP 监控模板，覆盖多类网络设备品牌与型号 |
| 监控中心 | 新增 HAProxy 监控能力，完善负载均衡场景监控覆盖 |
| 监控中心 | 支持批量配置告警策略模板，提升监控策略配置效率 |
| 日志中心 | 强化日志告警策略配置体验，支持告警预览，并支持在告警名称中引用变量 |
| 告警中心 | 新增告警自动化处置能力，可按规则自动触发作业并记录执行结果 |
| 告警中心 | 新增告警丰富（Enrichment）能力，支持基于 CMDB 资源等信息自动补充告警上下文 |
| 运营分析 | 内置告警分析大屏，帮助快速掌握告警趋势与处置情况 |
| 运营分析 | 新增网络拓扑组件，支持在分析视图中呈现网络关系 |
| 运营分析 | 分析视图支持接入系统管理操作日志 |
| OpsPilot | 工作台新增 NATS 触发节点，支持由告警中心触发 AI 工作流 |
