# April 24, 2026 Release Notes
Release Date: April 24, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Job Management | Supports downloading templates when uploading Playbooks. |
| CMDB | Added the ability to collect host configuration files. |
| Monitoring | Supports custom SNMP collection templates. |
| Monitoring | Added etcd monitoring capabilities. |
| Alert Center | Added a K8s Event ingestion channel. |
| OpsPilot | Tools now support configuring multiple instances. |
| OpsPilot | Added MySQL and Oracle tools. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Refactored the controller installation workflow to resolve installation timeouts caused by oversized packages. |
| Operations Analysis | Built-in analytics dashboards for Alert Center and CMDB are now available. |
