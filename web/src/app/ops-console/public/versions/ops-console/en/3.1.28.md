# 3.1.28 Release Notes
Release Date: May 15, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Supports automatically synchronizing hosts from Node Management into CMDB assets. |
| CMDB | Added SNMP and IPMI collection testing tools for quickly validating connectivity and configuration correctness. |
| Monitoring | Added a MySQL monitoring analysis view. |
| Monitoring | Kafka monitoring now supports Topic and Group filtering. |
| OpsPilot | Upgraded the React Agent to support Loop capabilities. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Alert events now support clicking a date to view the corresponding events. |
| Log Center | Adjusted the reporting node for Syslog and SNMP Trap to use the cloud region IP. |
| Log Center | Optimized the Docker log analysis view to improve the log viewing and analysis experience. |
