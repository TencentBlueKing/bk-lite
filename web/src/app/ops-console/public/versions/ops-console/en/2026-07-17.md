# July 17, 2026 Release Notes
Release Date: July 17, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Monitoring | Added an onboarding guide for monitoring objects to help users quickly complete monitoring configuration. |
| OpsPilot | NATS trigger node now supports a "Web Conversation" mode. |
| OpsPilot | Workflow test runs can now be manually stopped. |

### Improvements

| Module | Improvements |
| --- | --- |
| CMDB | Optimized CMDB collection task scheduling to reduce task backlog and duplicate execution risks. |
| CMDB | Improved the efficiency of large-scale asset synchronization and automatic association. |
| Monitoring | Optimized host monitoring onboarding: instance names are now auto-generated after selecting a collection node. |
| Monitoring | Optimized alert policy metric selection to display the original units of metrics. |
| Monitoring | Fixed an issue where special characters in IPMI monitoring passwords could not be properly escaped. |
| Alarm Center | Enhanced the configuration capabilities of action rules and related rules. |
| Alarm Center | Optimized the experience of alert details, field binding, and alert assignment. |
| Alarm Center | Fixed an issue with log alert integration. |
| Operations Analytics | Optimized the external data source preview mechanism to avoid performance impact from large data volumes. |
| Operations Analytics | Enhanced organization information display in the built-in canvas. |
| OpsPilot | Fixed an issue with skill pack hit and match exceptions. |
