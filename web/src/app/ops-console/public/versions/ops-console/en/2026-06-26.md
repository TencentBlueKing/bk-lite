# June 26, 2026 Release Notes
Release Date: June 26, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added a data center rack view with rack layout and U-space utilization display. |
| CMDB | Added IP Address Management (IPAM), supporting subnet planning, automatic IP discovery scans, and automatic reconciliation conflict management. |
| CMDB | Expanded configuration collection targets with new collection support for Huawei storage, MinIO, InfluxDB, and more. |
| Monitoring | Added 117 SNMP monitoring templates covering multiple network device brands and models. |
| Monitoring | Added HAProxy monitoring to improve coverage for load balancing scenarios. |
| Monitoring | Added support for batch configuring alert policy templates to improve monitoring policy setup efficiency. |
| Log Center | Enhanced log alert policy configuration with alert preview support and variable references in alert names. |
| Alert Center | Added automated alert handling that can trigger jobs by rule and record execution results. |
| Alert Center | Added alert enrichment, supporting automatic alert context supplementation based on CMDB resources and other information. |
| Operations Analytics | Added a built-in alert analysis dashboard to help quickly understand alert trends and handling status. |
| Operations Analytics | Added a network topology component for presenting network relationships in analysis views. |
| Operations Analytics | Analysis views now support access to system management operation logs. |
| OpsPilot | Added a NATS trigger node to the workbench, allowing Alert Center to trigger AI workflows. |
