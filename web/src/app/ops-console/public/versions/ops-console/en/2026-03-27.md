# March 27, 2026 Release Notes
Release Date: March 27, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Job Management | Supports job execution based on Ansible. |
| Log Center | Supports viewing Top value distributions through the field list on the left. |
| Alert Center | Added a Missing Detection strategy to detect expected events that did not arrive. |

### Improvements

| Module | Improvements |
| --- | --- |
| Platform | Optimized token expiry handling by showing a popup instead of redirecting directly to the login page. |
| Node Management | Optimized the managed component operation experience with visible execution steps and progress. |
| Alert Center | Optimized UTC time zone handling to improve alert notification accuracy. |
| OpsPilot | Optimized model onboarding interactions with batch management for models from the same vendor. |
