# July 3, 2026 Release Notes
Release Date: July 3, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Job Management | During job execution, the job details page now supports streaming display of execution progress and results. |
| CMDB | Added network device configuration collection for unified collection and management of network configurations. |
| CMDB | Added an application topology view to help quickly understand application relationships and dependency structures. |
| Monitoring | Added TCP probing to test port connectivity and service availability. |
| Monitoring | Monitoring lists now support custom display fields for different viewing scenarios. |
| Monitoring | The monitoring onboarding flow now supports connectivity tests to detect network, port, or authentication configuration issues early. |
| Operations Analytics | Expanded data source types with support for MySQL, PostgreSQL, REST API, and Excel. |
| OpsPilot | The workbench now supports integration with WeCom bots, making it easier to connect intelligent capabilities to enterprise collaboration scenarios. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Optimized monitoring view collection break-point detection, making abnormal data interruptions easier to locate. |
| OpsPilot | External notification nodes now support configuring LLM models to intelligently optimize notification content, making messages clearer and easier to understand. |
