# February 6, 2026 Release Notes
Release Date: February 6, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added constraint support for model attribute types: string supports regex validation; number supports maximum and minimum range constraints; time supports date and time formats. |
| CMDB | Model attributes now support user guidance text. |
| Monitoring | Added a no-data alert baseline mechanism to support precise dimension-level identification of no-data events. |
| OpsPilot | Logs now support viewing error causes and locating failed nodes. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Optimized the K8s metric system and added alert type display capabilities in the alert list. |
| Monitoring | Fixed issues such as incorrect SASL parameter construction for Kafka monitoring and incorrect handling of special characters in MySQL monitoring credentials. |
| OpsPilot | The browser_use tool now automatically captures key screenshots during browser operations, making agent actions visible. |
| OpsPilot | Configuration documentation for the PostgresDB, K8s, and Web application inspection assistants has been updated on the official website. |
