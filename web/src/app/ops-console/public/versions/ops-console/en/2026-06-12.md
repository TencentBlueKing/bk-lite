# June 12, 2026 Release Notes
Release Date: June 12, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| Node Management | Added support for remotely uninstalling manually installed nodes from the page. |
| Job Management | Job details now support viewing Playbook templates. |
| CMDB | Added the ability to report asset data through custom submissions. |
| CMDB | Enhanced network topology discovery capabilities. |
| Monitoring | Added metric configuration display capability to the monitoring view list. |
| Monitoring | Added WMI-based remote monitoring for Windows hosts. |
| Alert Center | Added an "alert escalation" option to alert assignment policies so unacknowledged alerts can be escalated automatically. |
| Operations Analytics | Dashboard table components now support configurable action columns and added drill-down capability. |
| OpsPilot | Added a Markdown file generation tool to support agents in producing structured documents. |
| OpsPilot | Email notification nodes now support sending agent-generated files as attachments. |

### Improvements

| Module | Improvements |
| --- | --- |
| Monitoring | Optimized monitoring view display logic to show only metrics from collected templates. |
| Log Center | Merged HTTP traffic and network traffic collection templates and exposed listening NIC configuration options. |
