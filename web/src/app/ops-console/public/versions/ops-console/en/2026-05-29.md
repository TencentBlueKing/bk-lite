# May 29, 2026 Release Notes
Release Date: May 29, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Supports manually uploading configuration files. |
| CMDB | Supports viewing configuration file content across different versions and comparing differences. |
| Monitoring | Alert notification channels now support multi-select. |
| Log Center | Added built-in log analysis views covering 15 common database and middleware scenarios. |
| Alert Center | Added alert level management for unified management of event, alert, and incident levels. |
| Alert Center | Added related alert recommendations, which automatically match and display associated alerts through correlation rules. |
| Operations Analysis | Supports displaying period-over-period changes in single-value, Top, and other views to help quickly identify trend fluctuations. |
| Operations Analysis | Added a built-in Monitoring Center dashboard for quickly understanding monitoring coverage. |
| OpsPilot | Added memory capabilities, supporting memory reads and writes in workflows to accumulate experience and reuse context. |

### Improvements

| Module | Improvements |
| --- | --- |
| System Management | Optimized the admin account password expiration mechanism: accounts are no longer locked directly after expiration, and users are guided to change the password during login. |
| Node Management | Fixed abnormal built-in Vector and Winlogbeat issues on Windows nodes. |
| Monitoring | Fixed SNMP monitoring exceptions on host nodes caused by a missing MIB translator. |
