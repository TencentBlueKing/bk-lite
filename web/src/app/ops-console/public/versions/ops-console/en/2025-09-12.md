# September 12, 2025 Release Notes
Release Date: September 12, 2025
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| OpsPilot | Added Chatflow applications, supporting multiple trigger modes including scheduled and API triggers. |
| Log Center | Added support for Network Flow and related log types. |
| Operations Analysis | Supports zooming, thumbnails, and component dragging. |

### Improvements

| Module | Improvements |
| --- | --- |
| OpsPilot | Upgraded RAG retrieval methods to support chunk-based and segment-based recall. |
| Log Center | Continued expanding Packetbeat log collection capabilities. |
| Operations Analysis | Improved the operations canvas interaction experience. |
