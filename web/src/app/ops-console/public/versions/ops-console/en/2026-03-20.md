# March 20, 2026 Release Notes
Release Date: March 20, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| CMDB | Added data subscriptions to send notifications when asset properties change or approach expiration. |
| CMDB | Added process collection for host acquisition. |
| Job Management | Launched the Job Management module, allowing scripts or file uploads to run on managed hosts from Node Management. |
| OpsPilot | Added built-in Redis and Elasticsearch operations tools. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Optimized the controller installation experience by showing step completion during installation. |
| Log Center | Log integrations have been comprehensively upgraded, providing rich scenario-based log ingestion capabilities. |
