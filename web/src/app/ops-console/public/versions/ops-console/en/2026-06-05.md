# June 5, 2026 Release Notes
Release Date: June 5, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| System Management | Built-in system administrator, security administrator, and audit administrator roles meet three-member management requirements. |
| CMDB | Added model layout management with support for category organization and drag-and-drop model sorting. |
| CMDB | Configuration collection tasks now support configuring multiple credentials. |
| Monitoring | Added remote host monitoring that can collect host metrics remotely without installing an agent. |
| Monitoring | Added built-in monitoring dashboards covering 17 categories of monitored objects, including databases and middleware. |
| Alert Center | Added an "instant alert" type to correlation rules so matched events can generate alerts immediately. |
| Operations Analytics | Added collapsible dashboard groups so complex dashboards can be folded and expanded by group. |
| Operations Analytics | Analysis views now support full-screen mode for a better large-screen display and data analysis experience. |

### Improvements

| Module | Improvements |
| --- | --- |
| OpsPilot | Optimized human-in-the-loop agent decision interactions to improve option confirmation and workflow progression. |
