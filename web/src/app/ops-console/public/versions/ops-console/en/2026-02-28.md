# February 28, 2026 Release Notes
Release Date: February 28, 2026
Learn more about product capabilities in the [official documentation](https://www.bklite.ai) or try the [Demo environment](https://bklite.canway.net).

### New Features

| Module | New Features |
| --- | --- |
| System Management | The Group Chat Assistant notification channel now supports Feishu, DingTalk, and other custom webhook bots. |
| OpsPilot | Logs now display agent reasoning steps. |

### Improvements

| Module | Improvements |
| --- | --- |
| Node Management | Fixed an issue where the Windows controller could not be installed correctly. |
| Monitoring | Optimized the alert compensation mechanism so Monitoring automatically backfills alert snapshots from abnormal periods after recovering from an outage. |
| Monitoring | Optimized alert name variables to support displaying metric dimension information at the time of the alert. |
| OpsPilot | Optimized scheduled trigger nodes in the workbench with cron expression support. |
