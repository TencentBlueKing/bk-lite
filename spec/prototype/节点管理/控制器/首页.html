Linux控制器（x86_64）

Linux控制器由 Sidecar 和 NATS Executor 组件组成，用于管理 Linux 节点和节点上的各类采集器，实现节点上的自动化部署、滚动调节与任务执行。控制器默认集成了 Telegraf 和 Node Exporter 的运行能力。

Linux x86_64

Windows控制器（x86_64）

Windows控制器由 Sidecar 和 NATS Executor 组件组成，用于管理 Windows 节点和节点上的各类采集器，实现节点上的自动化部署、滚动协调与任务执行。控制器默认集成了 Telegraf 相关运行时组件。

Windows x86_64

Linux控制器（ARM64）

Linux 控制器 ARM64 版本面向 ARM64 架构节点，提供与 Sidecar、NATS Executor 兼容的自动化部署与任务执行能力，适用于边缘节点、轻量服务器及异构计算场景下的采集器运行和统一管控。

Linux ARM64
