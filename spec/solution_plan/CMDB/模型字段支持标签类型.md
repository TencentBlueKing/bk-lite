# 模型字段支持标签类型

日期：2026-03-04

## 目标

在 CMDB 中新增内置标签字段能力（单模型单标签字段，字段 ID 固定为 `__tag__`），并以最小改动打通模型配置、实例录入/修改、列表展示、搜索、导入导出、批量更新全链路。

## 范围

- 模型字段定义与启停
- 标签字段编辑规则（自由/严谨模式）
- 实例新增/修改标签录入与校验
- 实例列表标签列展示
- 实例搜索（标签专用筛选）
- 实例导入导出与批量更新

## 已确认决策

- 严谨模式候选来源：仅模型字段配置中维护。
- 重复标签策略：保存前统一去重。
- 关闭标签开关审计：使用独立审计事件，与普通删字段区分。
- 标签搜索二层树数据源：基于模型字段候选项，不基于实例实时聚合。

## 方案概览

采用“后端类型扩展 + 统一校验内核 + 前端最小适配 + 全链路复用”的渐进式方案：

1. 在现有字段类型体系中新增 `tag` 类型，保持当前模型/实例主流程不拆分。
2. 在模型字段中引入内置标签开关，并约束“单模型最多 1 个标签字段、ID 固定 `__tag__`”。
3. 抽出统一标签解析与校验逻辑，供实例新增、实例修改、批量更新、导入共用，避免规则漂移。
4. 前端复用现有字段渲染与搜索框架，仅对 `tag` 类型增加专用控件与交互。
5. 导入导出沿用现有流程，新增标签格式解析与错误定位，不引入新服务或额外中间件。

## 数据与规则约定

### 1）字段定义约束

- 标签字段 `attr_id` 固定为 `__tag__`，不可编辑。
- 标签字段名称、属性分组可编辑。
- 内置约束固定不可改：可编辑=是，唯一=否，必填=否。
- 单模型最多启用 1 个标签字段。

### 2）标签值语义

- 单个标签格式：`key:value`。
- `key` 不允许为空。
- `value` 不允许包含空格、冒号、换行符。
- 支持同 key 多 value。
- 单实例最多 20 个标签对。
- 保存前按完整 `key:value` 去重。

### 3）模式语义

- 自由模式：允许输入模型未定义候选标签。
- 严谨模式：仅允许选择模型已定义候选标签，不允许自定义。

### 4）导入导出分隔符

- 多标签对支持：英文逗号（`,`）、中文逗号（`，`）、换行。
- 导入、导出、批量更新复用同一标签校验规则。

## 分阶段计划

### 阶段一：后端基础能力落地

**交付内容**
- 新增 `tag` 字段类型定义与类型映射。
- 模型字段开关与单模型唯一标签字段约束。
- 关闭开关时物理删除标签字段与关联标签数据。
- 增加独立审计事件记录“标签字段关闭删除”。

**阶段验收**
- 模型可启用/关闭标签字段。
- 单模型无法创建第二个标签字段。
- 关闭后实例相关接口不再返回标签字段。

### 阶段二：统一校验与实例链路打通

**交付内容**
- 新增标签统一解析/校验逻辑（格式、字符、数量、去重）。
- 接入实例新增、实例修改、批量更新、导入流程。
- 严谨模式下增加候选合法性校验；自由模式放开候选限制。

**阶段验收**
- 四条写入链路执行一致校验规则。
- 非法标签返回可读错误信息。
- 超过 20 个标签对保存失败并提示。

### 阶段三：搜索与展示适配

**交付内容**
- 标签专用二层树筛选组件（key/value），支持多选回显。
- 同一查询会话跨页保留已选；切换筛选条件清空已选。
- 标签搜索固定精确匹配，`Accurate` 固定开启。
- 列表标签列展示：逗号分隔、超长截断、tooltip 完整查看。

**阶段验收**
- 标签搜索仅精确匹配且不可切换模糊。
- 二层筛选按模型候选项返回并可正常多选过滤。
- 列表展示符合截断与 tooltip 要求。

### 阶段四：导入导出联调与发布准备

**交付内容**
- 导入兼容三类分隔符并执行统一校验。
- 导出标签格式统一为 `key:value` 序列。
- 错误信息可定位（至少包含行号、字段名）。
- 回归验证与上线回滚预案。

**阶段验收**
- 导入、导出、批量更新行为一致。
- 非法导入数据可准确定位并阻断入库。
- 通过模块最小门禁后具备上线条件。

## 验收清单

- 单模型可启用且仅可保留 1 个标签字段，字段 ID 固定为 `__tag__`。
- 关闭标签开关后，标签字段与关联标签数据被物理删除，相关页面与能力链路不再出现标签。
- 标签字段编辑时 ID 不可改，名称与分组可改，内置约束不可改。
- 自由模式允许自定义标签；严谨模式仅允许已定义候选。
- 同 key 多 value 可保存与回显；保存前去重生效。
- 实例新增/修改、批量更新、导入均执行格式与“最多 20 对”校验。
- 标签列表列超长截断并可 tooltip 查看完整内容。
- 标签搜索使用专用二层筛选，支持多选回显与同会话跨页保留。
- 标签搜索仅精确匹配，`Accurate` 固定开启。
- Excel 导入导出支持 `key:value`，并兼容英文逗号、中文逗号、换行分隔。

## 风险与应对

- 风险：多入口各自实现标签校验，导致规则不一致。  
  应对：抽取统一标签校验函数，所有写入入口强制复用。

- 风险：关闭标签字段时数据删除不完整，产生脏数据。  
  应对：字段删除与关联数据删除放入同一事务流程，失败即整体回滚。

- 风险：严谨模式候选项维护不当导致录入失败率升高。  
  应对：模型配置页提供清晰候选维护入口与错误提示，录入侧提示具体不合法项。

- 风险：标签搜索组件状态管理复杂导致跨页选择异常。  
  应对：复用现有查询会话状态容器，新增最小状态字段并补充回归用例。

## 非目标（本期不做）

- 标签三级及以上层级结构。
- 标签 value 模糊检索能力。
- 标签自动推荐、标签治理统计等运营能力。
- 非标签类型字段的同步改造。
