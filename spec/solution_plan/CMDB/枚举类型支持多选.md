# 枚举类型支持多选

日期：2026-03-10

## 方案概览

本方案基于 PRD 与 Requirement，面向 CMDB `enum` 字段新增“单选/多选”选择模式能力，覆盖模型字段配置、实例录入与编辑、列表展示、搜索筛选、批量更新、导入导出全链路。

总体策略：
- 严格限定改造范围：仅 `enum` 字段生效，不扩展到其他字段类型。
- 渐进式落地：复用现有模型与实例链路，不新增页面体系与复杂中间层。
- 兼容优先：默认行为保持单选，存量字段不自动迁移。
- 规则前置：字段创建后锁定选择模式，避免运行期语义漂移。

## 范围与约束

### In Scope

- 枚举字段配置新增“选择模式”：`single` / `multiple`，默认 `single`。
- 选择模式仅存储在模型字段 `attr` 属性下。
- 实例新增、编辑、复制、详情编辑支持多选枚举录入。
- 列表展示支持多值展示（英文逗号分隔）。
- 搜索筛选支持多值筛选（字段内 OR、字段间 AND）。
- 批量更新支持多选枚举赋值。
- 导出统一英文逗号；导入支持英文逗号、中文逗号、换行分隔。

### Out of Scope

- 非 `enum` 字段类型能力改造。
- 旧字段自动数据迁移与历史值自动修复。
- 新增审批流、版本管理、跨租户共享等增强能力。

## 已确认决策

- 存储口径：`enum_select_mode` 仅存模型字段 `attr` 属性，不新增独立持久化字段。
- 搜索语义：字段内 OR、字段间 AND。
- 导入导出口径：导出英文逗号；导入支持英文逗号、中文逗号、换行，并按模式做数量校验。
- 导入失败策略：采用“全量错误汇总”后一次返回。

## 分阶段计划

### 阶段一：字段配置契约落地

**里程碑目标**
- 完成枚举字段“选择模式”配置能力与后端契约对齐。

**交付内容**
- 前端字段配置弹窗新增并默认选择 `single`。
- 后端模型字段保存与回显透传 `enum_select_mode`（落入模型字段 `attr` 属性）。
- 编辑已有字段时，模式控件锁定不可切换。
- 历史字段无模式值时按 `single` 解释。

### 阶段二：实例写入与展示一致性

**里程碑目标**
- 多入口写入与展示遵循同一模式语义。

**交付内容**
- 实例新增/编辑/复制/详情编辑：
  - `single`：仅允许 1 个值。
  - `multiple`：允许多个值（去空、去重后校验）。
- 列表展示与详情展示：多值统一英文逗号拼接输出。
- 批量更新复用同一校验能力，保证与表单提交口径一致。

### 阶段三：查询与导入导出打通

**里程碑目标**
- 完成多选枚举在筛选与数据交换链路的端到端闭环。

**交付内容**
- 搜索筛选支持多值传参，查询执行“字段内任一命中”。
- 导出统一使用英文逗号序列化多值。
- 导入支持 `,` / `，` / 换行解析；并按模式校验值数量与合法性。
- 导入失败采用全量错误汇总后一次返回，并可定位错误（行号、字段、非法值）。

### 阶段四：联调、灰度与发布

**里程碑目标**
- 低风险上线并具备可回退路径。

**交付内容**
- 前后端联调覆盖 Requirement 全场景。
- 先小范围灰度，观察保存失败率、导入失败率、筛选命中准确性。
- 回滚策略：回退到单选解析分支（解析与校验层回退，不改历史数据）。

## 阶段验收清单

- 可新建枚举字段并选择单选/多选，默认单选。
- 已创建字段编辑时，选择模式不可修改。
- 多选字段在实例新增/编辑/复制/详情编辑可正确写入与回显。
- 列表中多值以英文逗号分隔展示。
- 筛选支持多选值，结果满足“任一命中”。
- 批量更新可对多选字段写入并通过统一校验。
- 导出文件中多值字段为英文逗号分隔。
- 导入支持英文逗号、中文逗号、换行；错误可定位到行和字段。
- 存量单选字段行为不变，无强制迁移。

## 风险与应对

- 风险：各入口（表单/批量/导入）校验口径不一致，导致脏数据。  
  应对：统一复用后端同一枚举校验函数，禁止分叉实现。

- 风险：字段模式被误改引发历史数据解释混乱。  
  应对：服务端强约束“创建后不可切换”，前端仅做禁用展示。

- 风险：导入分隔符多样导致解析歧义。  
  应对：固定解析优先级并输出标准化错误信息（行号/字段/非法值）。

- 风险：筛选语义与用户预期偏差。  
  应对：在需求与联调用例中明确“字段内 OR、字段间 AND”并回归验证。

## 依赖与协同

- 产品：确认模式文案与错误提示口径。
- 后端：字段契约、统一校验、查询条件、导入导出解析。
- 前端：字段配置交互、实例输入组件、列表与筛选适配。
- 测试：覆盖新增/编辑/复制/筛选/批量/导入导出全链路。
