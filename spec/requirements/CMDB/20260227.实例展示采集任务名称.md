# 实例展示采集任务名称

日期：2026-02-27

## 背景

实例字段 `collect_task` 存放采集任务 ID，当前在实例列表与详情页直接展示 ID，理解成本高。

## 需求描述

在实例列表与实例详情页，将 `collect_task` 字段展示为采集任务名称 + ID，格式为：

```
{任务名称}({id})
```

## 适用范围

- 实例列表
- 实例详情

## 展示规则

- 若 `collect_task` 为空，按空值展示规则处理（如：空/占位符）。
- 若 `collect_task` 有值且能匹配到采集任务名称，按 `{任务名称}({id})` 展示。
- 若 `collect_task` 有值但未找到对应任务，展示 `未找到任务(id)`。

## 验收标准

- 实例列表与实例详情页中，`collect_task` 均按 `{任务名称}({id})` 展示。
- 当任务不存在时，显示 `未找到任务(id)`。
- 空值不展示错误信息，不影响其他字段显示。
