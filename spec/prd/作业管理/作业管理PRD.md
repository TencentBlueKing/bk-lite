# 作业管理 PRD

版本：v1.1

日期：2026-05-28

## 1. 背景与目标

作业管理是平台统一的脚本执行、文件分发与作业调度中心，围绕“作业准备 -> 执行 -> 追踪 -> 复用 -> 治理”五类核心场景提供稳定能力。

目标：
- 提供统一入口覆盖快速执行、文件分发、定时调度与执行记录。
- 通过目标管理统一维护异构主机的连接驱动与凭据。
- 通过脚本库与 Playbook 库实现模板复用与参数化执行。
- 通过高危命令与高危路径规则降低误操作风险。

## 2. 范围定义

### 2.1 在范围内

- 首页总览：作业能力入口卡片与最近作业记录。
- 作业执行：快速执行、文件分发。
- 定时任务：任务列表、任务创建与编辑、任务启停。
- 作业记录与作业详情（脚本、Playbook、文件分发）。
- 作业模板：脚本库、Playbook 库。
- 目标管理：目标维护、驱动与凭据配置、连接测试。
- 系统配置：高危命令配置、高危路径配置。
- 对外接口：第三方 API 触发作业并支持执行完成回调。

### 2.2 不在范围内

- 第三方审批流与工单流转。
- 跨产品线的统一告警中心。
- 非作业管理模块的资产模型维护能力。

## 3. 用户与角色

- 运维工程师：执行与调度作业、查看执行日志与结果。
- 平台管理员：维护目标、模板与安全规则。
- 安全管理员：维护高危拦截规则与组织级生效范围。

## 4. 关键能力

### 4.1 首页

- 展示作业能力入口卡片（快速执行、文件分发、定时任务）。
- 展示最近作业记录，包含作业名称、作业类型、触发来源、执行状态、开始时间。

### 4.2 目标管理

- 目标列表支持按名称、IP、云区域、来源、驱动、组织搜索。
- 支持手动添加目标与从节点管理同步目标。
- 目标字段包含：名称、IP、操作系统、云区域、来源、当前驱动、凭据、组织。
- 支持目标编辑与删除。
- 执行驱动支持：Ansible、SSH、Sidecar。
- 凭据来源支持：手动录入、凭据管理。
- 手动录入支持 Linux 的 SSH 连接（密码或密钥认证）与 Windows 的 WinRM 连接。
- 支持连接测试。

### 4.3 脚本库

- 脚本列表展示：脚本名称、脚本类型、组织、创建人、更新时间、描述。
- 支持新建脚本，脚本类型支持：Shell、Bat、Python、PowerShell。
- 支持脚本内容编辑，内容支持 Jinja2 参数模板语法。
- 支持参数定义管理（新增、编辑、删除），参数项包含参数名、标签、是否加密、默认值、提示信息。
- 内置脚本允许查看与引用。

### 4.4 Playbook 库

- Playbook 列表展示：名称、版本、组织、更新时间、描述。
- 支持上传 Playbook 与版本更新。
- 支持查看详情、下载、修改基础信息、删除。
- 详情页支持查看参数定义（参数名、默认值、说明）。

### 4.5 快速执行

- 创建即时作业，字段包含：作业名称、目标主机、内容来源、模板或脚本内容、执行参数、超时时间。
- 内容来源支持：作业模板、临时输入。
- 模板类型支持：脚本库、Playbook。
- 临时输入支持脚本语言切换：Shell、Bat、Python、PowerShell。
- 支持目标主机弹窗选择、搜索、批量勾选。
- 支持立即执行。

### 4.6 文件分发

- 创建文件分发作业，字段包含：作业名称、目标主机、上传文件、目标路径、覆盖策略、超时时间。
- 文件上传支持文件与压缩包。
- 覆盖策略支持：覆盖已存在文件、跳过已存在文件。
- 支持目标主机弹窗选择、搜索、批量勾选。
- 支持立即执行。

### 4.7 定时任务

- 任务列表展示：任务名称、作业类型、执行策略、启停状态、下次执行时间、操作。
- 支持新建与编辑定时任务。
- 必填信息：任务名称、目标主机、作业类型、模板或内容、执行策略。
- 作业类型支持：脚本执行、文件分发。
- 执行策略支持简单策略与复杂策略：简单策略支持每天一次、每 N 小时一次、只执行一次；复杂策略支持 Cron 字段编辑（分、时、日、月、周）、表达式描述与下次执行时间预览。
- 支持并发策略与超时时间配置。
- 支持任务启停切换。
- 支持“保存并启用”与“仅保存”。
- 支持列表筛选与搜索。

### 4.8 作业记录

- 列表展示：作业 ID、作业名称、作业类型、触发来源、状态、发起人、开始时间、耗时。
- 支持按状态、类型、来源与时间范围筛选。
- 支持时间快捷筛选（今天、近 7 天、近 30 天、自定义）。
- 支持分页。

### 4.9 作业详情

- 通用详情头：作业标题、作业 ID、状态、重新执行。
- 基础信息区展示：作业类型、触发来源、发起人、开始时间、耗时、目标主机、超时时间及作业相关扩展字段。
- 主机维度执行结果展示：每台主机状态与耗时。
- 执行日志能力：日志搜索、复制、下载（脚本详情含自动滚动）。
- 脚本作业详情：支持查看脚本抽屉，展示脚本来源、语言、执行用户、代码行数与脚本内容。
- Playbook 作业详情：支持查看 Playbook 抽屉，展示来源、名称、版本、代码行数与内容，并支持按主机查看步骤执行状态与日志。
- 文件分发详情：支持查看分发文件抽屉，展示传输详情（文件路径、传输进度、速率、完成状态、总量统计）。

### 4.10 高危命令配置

- 规则列表展示：规则名称、命令匹配规则、处理策略、组织、启停、更新时间、操作。
- 支持按规则名称、匹配规则、处理策略、组织、状态搜索。
- 支持新增、编辑、删除规则，规则字段包含规则名称、匹配规则、处理策略、组织、描述。
- 命令匹配规则支持正则表达式。
- 处理策略支持：禁止执行、二次确认。
- 支持规则启停。

### 4.11 高危路径配置

- 规则列表展示：规则名称、路径匹配规则、匹配方式、处理策略、组织、启停、更新时间、操作。
- 支持按规则名称、匹配规则、处理策略、组织、状态搜索。
- 支持新增、编辑、删除规则，规则字段包含规则名称、路径匹配规则、匹配方式、处理策略、组织、描述。
- 匹配方式支持：精确匹配、正则匹配。
- 处理策略支持：禁止分发、二次确认。
- 支持规则启停。

## 5. 关键规则

### 5.1 作业类型与触发来源

- 作业类型包含：脚本执行、文件分发、Playbook。
- 触发来源包含：手动执行、定时任务、API 调用。

### 5.2 主机驱动与执行兼容性

- 目标主机驱动包含：Ansible、SSH、Sidecar。
- 当执行内容为 Playbook 时，仅允许选择 Ansible 驱动的目标主机。
- 不兼容主机会在选择列表中禁用并提示原因。

### 5.3 目标来源约束

- 节点管理同步来源的目标，编辑时基础信息字段不可修改。
- 手动录入来源的目标，允许完整编辑基础信息。

### 5.4 凭据规则

- 节点管理同步来源的目标在本地节点执行，无需额外连接凭据。
- 手动录入来源的目标按操作系统选择凭据：Linux 使用 SSH，Windows 使用 WinRM。
- SSH 凭据支持密码与密钥两种认证方式。
- 凭据可手动录入或来自凭据管理。

### 5.5 安全拦截规则

- 高危命令规则作用于命令执行场景。
- 高危路径规则作用于文件分发路径场景。
- 处理策略为“禁止”时阻断执行；为“二次确认”时需确认后继续。
- 规则支持组织范围生效。
- 规则状态仅包含：已启用、已停用。

### 5.6 定时调度规则

- 任务支持启用与停用两种状态。
- 启用任务需显示下次执行时间。
- 并发策略支持：跳过、运行、排队。

### 5.7 执行追踪规则

- 执行记录保留作业元信息、参数快照、执行用户与主机维度结果。
- 执行状态包含：等待中、执行中、成功、失败、超时、已取消。
- 队列中的作业支持取消，已完成作业支持重新执行。
- API 触发的作业可在执行完成后回调外部地址。

## 6. 非功能要求

- 可用性：关键流程（快速执行、文件分发、定时任务新建）保持单页可完成。
- 可追溯性：作业详情完整保留执行元信息与日志。
- 安全性：高危规则可配置且支持组织级控制；加密参数与凭据不明文回显。

## 7. 验收标准

- 用户可从首页进入任一作业能力并完成一次操作。
- 用户可在目标管理完成新增目标、同步目标、编辑目标、删除目标与连接测试。
- 用户可在脚本库和 Playbook 库完成模板维护，并在执行页面引用模板。
- 用户可完成一次快速执行与一次文件分发作业创建。
- 用户可完成一次定时任务创建并在列表看到启停状态与下次执行时间。
- 用户可在作业记录中按状态与时间范围筛选并进入详情。
- 用户可在作业详情中按主机查看状态与日志，并执行日志复制与下载。
- 用户可新增并启用高危命令与高危路径规则，并在列表中看到规则状态变化。

## 8. 边界与约束

- 高危命令与高危路径规则当前不提供系统内置只读规则。
- 高危路径匹配仅支持精确匹配与正则匹配，不支持通配符与多模式分隔。
- Playbook 执行依赖 Ansible 驱动主机，其他驱动不支持 Playbook。
