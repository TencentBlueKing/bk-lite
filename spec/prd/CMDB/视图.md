# CMDB · 视图

## 1. 背景与目标

视图提供资产总览入口，按分类聚合展示各模型的资产数量，帮助用户快速了解资产分布并跳转到对应资产列表。

## 2. 范围定义

- 资产总览：按分类聚合展示模型及其资产数量。
- 模型筛选：按模型名快速过滤。
- 快速跳转：从模型卡片进入对应资产列表。

## 3. 关键能力

- 按分类聚合展示全部模型，每个模型卡片显示图标、名称和实时资产数量。
- 支持按模型名做大小写不敏感的子串过滤；过滤后空分类自动隐藏。
- 点击模型卡片携带模型与分类标识跳转到资产列表。

## 4. 关键规则

- 资产数量与可见模型受用户组织权限范围约束。
- 视图为全局统一结构，不提供个性化保存或置顶。

## 5. 关键技术架构选择

- 资产数量按模型从图数据库 FalkorDB 统计，与分类树合并后呈现。

## 6. 验收标准

- 用户可按分类查看各模型的实时资产数量。
- 可按模型名过滤并从卡片跳转到对应资产列表。
- 数量与可见范围正确反映用户权限。
