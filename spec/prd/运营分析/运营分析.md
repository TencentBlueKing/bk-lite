# 运营分析 · 运营分析

## 1. 背景与目标

运营分析提供可视化分析视图，让用户以仪表板、拓扑、架构图等形式自助编排和查看跨来源的运营数据。

## 2. 范围定义

- 视图类型：仪表板、拓扑、架构图。
- 视图组织：自定义目录组织视图。
- 过滤与命名空间：统一过滤条与命名空间选择。

## 3. 关键能力

- 支持三类视图，按自定义目录分层组织：
  - 仪表板：栅格化拖拽布局，支持折线、柱状、饼图、表格、单值、Top-N、仪表盘（gauge）、分级条形仪表盘（barGauge）、状态时间线（stateTimeline）、文本（text）、事件表（事件流）共 11 类组件，每个组件绑定数据源查询。
  - 拓扑：图形可视化，节点可展示单值或图表并实时刷新，支持节点与边。
  - 架构图：等距图编辑，可拖入云厂商、K8s、资产等图标设计系统拓扑。
- 视图通过数据源（在管理中配置）取数，查询支持固定参数、用户筛选与运行时参数。
- 提供统一过滤条（下拉、时间范围、单值），过滤项绑定组件查询并作用于整个视图。
- 提供命名空间选择，决定视图取数的来源连接。
- 支持编辑模式（增删/重排组件、改查询）与自动保存、未保存离开提醒。
- 仪表板支持可折叠分组：编辑态可新增、重命名、取消分组、删除整组，分组可整体拖拽排序与展开/收起；查看态保留展开/收起。
- 单值、Top-N、仪表盘组件支持周期对比：开启后自动对比上一个等长周期的数据并显示变化百分比；支持按阈值区间为数值/指针配色。
- 仪表盘组件支持半圆与整圆两种形态，可配置最小值、最大值与阈值配色刻度；分级条形仪表盘（barGauge）复用相同的阈值与量程配置。
- 状态时间线（stateTimeline）组件支持配置值映射规则，将原始值映射为展示文本与颜色区段。
- 文本（text）组件支持输入静态文本内容（轻量 Markdown），用于在仪表板中嵌入说明或标题。
- 折线图与柱状图组件支持堆叠系列开关（stack），开启后多系列数据堆叠展示。
- 单值、仪表盘、拓扑节点等支持值映射（Value Mappings）：可配置 value / range / regex / special 四类规则，按声明顺序首条命中，将原始值映射为自定义展示文本与颜色，覆盖默认数值显示。
- 组件单位支持结构化单位库（对齐 Grafana）：bytesIEC / bytesSI / bps / ms / percent / percentunit / short / none 等，选择后自动量纲缩放（如字节自动换算 KiB/MiB/GiB）；未选结构化单位时可继续输入自由文本后缀（向后兼容）。
- 事件表（事件流）组件按配置展示字段列表，支持分页与行展开查看明细。
- 三类视图均支持全屏查看；拓扑额外支持大屏配置：设定固定分辨率（宽、高）与大屏背景色（信箱填充色），并将该视口随视图一同持久化、支持导入导出。

## 4. 关键规则

- 视图按组织分组，用户仅可见所属组织的视图。
- 内置视图对组织可见但常规用户不可删改。
- 组件不直接查询其他模块，统一经数据源取数。
- 仪表盘组件最大值必须大于最小值，否则配置不生效。
- 阈值配色按阈值升序划分区间，命中区间的颜色用于数值显示与刻度/指针着色；barGauge 与 gauge 共用同一套阈值配置逻辑。
- 周期对比仅在数据源已配置可识别的时间范围参数时可开启，否则不可用。
- 事件表分页依赖数据源是否提供分页参数；未配置展示字段时组件不渲染数据。
- 拓扑大屏视口仅在同时填写宽与高时生效；背景色仅接受合法十六进制颜色，否则回退为黑色。
- 值映射规则按声明顺序首条命中即生效，后续规则不再匹配；无命中时沿用原始值展示。
- 结构化单位与自由文本后缀互斥：选择了结构化 unitId 后，自由文本 unit 字段隐藏；清空 unitId 则回退显示自由文本字段，旧配置不受影响。
- 非法正则规则（regex 类型）不抛错，自动忽略该条规则继续匹配后续规则。

## 5. 关键技术架构选择

- 模块定位为"可视化 + 配置层"，本身不做数据转换或聚合，取数全部下沉到用户定义的 REST 数据源。
- 取数经 NATS 按命名空间路由到下游服务；请求携带用户上下文用于下游鉴权。
- 视图配置支持 YAML 导入导出，并解析依赖（如仪表板带出其数据源）；拓扑导入导出含节点、边、过滤项与视口配置。

## 6. 验收标准

- 用户可创建并查看仪表板、拓扑、架构图三类视图。
- 组件可绑定数据源并按过滤条、命名空间取数。
- 视图按组织范围正确隔离，内置视图不可删改。
- 仪表板可创建可折叠分组并对组件进行分组、取消分组、删除整组，分组与组件均可拖拽排序。
- 仪表盘、单值、Top-N 组件可按阈值配色，开启周期对比后可显示较上周期的变化百分比。
- barGauge 组件可配置阈值与量程，与 gauge 共享配置面板。
- stateTimeline 组件可配置值映射规则，原始值命中规则后显示映射文本与颜色区段。
- text 组件可输入静态文本内容并在仪表板中渲染展示。
- 折线图、柱状图组件可开启堆叠系列开关，多系列数据堆叠展示。
- 支持值映射的组件（单值、仪表盘、stateTimeline、拓扑节点）可添加 value/range/regex/special 四类规则，命中后展示映射文本与颜色。
- 组件单位可选结构化单位（bytesIEC/bytesSI/bps/ms/percent/percentunit/short/none），数值自动按量纲缩放展示；也可输入自由文本后缀（兼容旧配置）。
- 事件表组件可按配置字段展示数据，支持分页与行展开查看明细。
- 三类视图可进入全屏；拓扑可设置固定分辨率与大屏背景色，并随视图导入导出保留。
