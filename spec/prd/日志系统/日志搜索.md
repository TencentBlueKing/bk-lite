# 日志系统 · 日志搜索

## 1. 背景与目标

日志搜索面向问题排查场景，提供跨日志分组的全文检索、实时跟踪与字段统计，帮助用户快速定位日志。

## 2. 范围定义

- 检索：按查询语句与时间范围检索日志。
- 实时跟踪：日志实时追踪。
- 统计：按时间与字段聚合、按字段 Top-N 统计。
- 字段发现：自动提取可用字段及其取值分布。
- 保存检索：保存与复用检索条件。

## 3. 关键能力

- 支持按查询语句（支持通配）跨所选日志分组检索，可按字段过滤。
- 支持灵活时间范围（默认最近 15 分钟）。
- 支持日志实时跟踪、按时间与字段的命中聚合、按字段 Top-N 统计、原始日志查看。
- 支持自动提取可用字段并展示时间范围内的字段取值分布。
- 支持按组织创建、编辑、删除可复用的检索条件（含查询语句、时间范围、日志分组选择）。

## 4. 关键规则

- 检索必须选择一个或多个日志分组，分组受权限控制。
- 检索与保存的条件按组织隔离。

## 5. 关键技术架构选择

- 日志统一存储与检索基于 VictoriaLogs，查询语句翻译为 VictoriaLogs 查询语言。
- 原始日志数据以压缩 JSON 存于对象存储。

## 6. 验收标准

- 用户可按查询语句与时间范围跨分组检索日志并实时跟踪。
- 可按字段做聚合与 Top-N 统计，并发现可用字段。
- 检索条件可保存复用，并按组织正确隔离。
