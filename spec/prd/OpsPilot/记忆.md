# OpsPilot · 记忆

## 1. 背景与目标

记忆为对话提供跨会话的长期上下文（用户画像、偏好、历史交互），让智能体的回答更连贯、更个性化。

## 2. 范围定义

- 记忆空间：记忆的容器与可见范围。
- 记忆条目：记忆内容的存储。
- 配置：默认模型与写入规则。

## 3. 关键能力

- 记忆空间作为对话记忆容器，可设可见范围（个人/团队）与写入规则。
- 记忆条目以标题 + 内容形式按记忆空间存储。
- 配置默认模型、写入规则（自动存储的判定条件）与可见范围。
- 用于在多轮对话间保留长期上下文。

## 4. 关键规则

- 记忆按个人或团队范围隔离。
- 写入规则决定哪些信息被自动沉淀为记忆。

## 5. 关键技术架构选择

- 记忆作为对话上下文的长期存储，按记忆空间组织并按规则写入。

## 6. 验收标准

- 用户可创建记忆空间并配置可见范围与写入规则。
- 记忆条目可按空间存储并在对话中提供长期上下文。
