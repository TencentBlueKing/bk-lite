# OpsPilot · 模型

## 1. 背景与目标

模型管理各类模型提供商与模型实例，为智能体、知识库等提供 LLM、嵌入、重排与 OCR 能力，是 OpsPilot 的算力底座。

## 2. 范围定义

- 模型类型：LLM、嵌入、重排、OCR。
- 提供商：多厂商接入与配置。
- 模型实例：模型的注册与启停。

## 3. 关键能力

- 支持四类模型：LLM、嵌入、重排、OCR。
- 厂商支持多种（如 OpenAI、Azure、阿里、智谱、百度、Anthropic、DeepSeek 及自定义 OpenAI 兼容），协议兼容 OpenAI 与 Anthropic。
- 配置 API 密钥（加密）、接入地址与模型名映射，支持系统预置默认模型。
- 各类模型分工：嵌入模型用于知识切分与检索，重排模型用于检索结果重排，OCR 模型用于文档取文。

## 4. 关键规则

- 模型按组织隔离，组织只能看到被授予的模型。
- API 密钥加密存储。

## 5. 关键技术架构选择

- 以提供商 + 模型实例的方式统一接入多厂商模型，协议层兼容 OpenAI/Anthropic。

## 6. 验收标准

- 用户可接入并配置 LLM、嵌入、重排、OCR 各类模型。
- 模型按组织隔离，密钥加密存储。
- 各类模型可被智能体与知识库正确选用。
