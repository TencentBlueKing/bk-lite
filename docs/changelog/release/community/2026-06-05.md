### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 内置系统管理员、安全管理员、审计管理员角色，满足三员管理要求 |
| CMDB | 新增模型布局管理能力，支持分类与模型拖拽排序 |
| CMDB | 配置采集任务支持配置多凭据 |
| 监控中心 | 新增远程主机监控能力，无需安装 Agent 即可远程采集主机监控数据 |
| 监控中心 | 新增内置监控仪表盘，覆盖数据库、中间件等 17 类监控对象 |
| 告警中心 | 相关性规则新增“即时告警”类型，命中事件可立即生成告警 |
| 运营分析 | 仪表盘新增可折叠分组能力，复杂看板可按分组收起与展开 |
| 运营分析 | 分析视图支持全屏查看，提升大屏展示与数据分析体验 |
