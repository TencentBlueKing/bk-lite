# 3.1.29 版本日志
版本发布时间：2026年5月22日
了解更多产品能力，欢迎查阅[官方文档](https://www.bklite.ai)或直接体验[Demo环境](https://bklite.canway.net)

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 新增用户操作：禁用、启用、解锁（密码输入次数超限后解锁） |
| CMDB | 自动发现页面新增采集任务概览统计 |
| 监控中心 | 新增华为 OceanStor 存储监控能力 |
| 告警中心 | Event 集成新增归属组织定义能力 |
