# 3.0 版本日志
版本发布时间：2025年9月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 新增 腾讯蓝鲸认证源，支持与蓝鲸平台单点登录，并可同步用户信息 |
| OpsPilot | Chatflow 应用支持 测试功能，可及时反馈测试结果 |
| OpsPilot | Chatflow 应用新增 历史记录展示 |
| 日志中心 | 日志告警事件列表 新增 时间分布图 |
| 运营分析 | 支持 2.5D 拓扑图，可绘制全应用架构 |
| OpsPilot | Chatflow 应用继续优化，支持完整设置流程类应用，并支持 定时触发 / API 触发 |
| OpsPilot | 智能体新增 问题优化 ，可对用户提问通过 LLM 进行补充和优化 |
| 日志中心 | 新增 日志分析，内置 网络流量基础仪表盘 |
| 日志中心 | 支持 保存日志搜索条件 |
| OpsPilot | 新增 Chatflow 类应用，支持 定时触发 / API 触发 等多种方式 |
| OpsPilot | RAG 召回方式升级，支持 分块召回 / 分段召回 |
| 日志中心 | 持续扩展 Packetbeat 日志采集能力 |
| 日志中心 | 新增支持 Network Flow 等类型 |
| 运营分析 | 支持放大缩小、缩略图、组件拖动操作 |
