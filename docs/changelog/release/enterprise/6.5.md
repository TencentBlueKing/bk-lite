# 6.5 版本日志
版本发布时间：2026年2月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 通知渠道-群聊小助手 新增支持接入飞书、钉钉及其他自定义的webhook机器人 |
| OpsPilot | 日志新增智能体思考步骤展示 |
| CMDB | 模型属性类型新增约束条件支持： string 支持正则校验 number 支持最大 / 最小值范围约束 time 支持日期与时间格式 |
| CMDB | 模型属性支持添加用户提示信息 |
| 监控中心 | 新增无数据告警基准机制，支持维度级别精准识别无数据事件 |
| OpsPilot | browser_use工具执行浏览器操作时自动生成关键截图，可视化查看 Agent 操作过程 |
| OpsPilot | 日志支持查看错误原因并定位错误节点 |
| OpsPilot | PostgresDB / K8s / Web 应用巡检助手配置操作文档已同步更新至官网 |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 节点管理 | 修复Windows控制器无法正常安装的问题 |
| 监控中心 | 优化告警补偿机制，监控中心从异常恢复后，自动补全异常期间的告警快照 |
| 监控中心 | 优化告警名称变量，支持展示告警时的指标维度信息 |
| OpsPilot | 优化工作台定时触发节点功能，支持cron表达式配置 |
| 监控中心 | 优化 K8s指标体系，告警列表展示告警类型 等能力 |
| 监控中心 | 修复 Kafka监控SASL参数构建错误、MySQL监控凭据特殊字符处理错误 等问题 |
