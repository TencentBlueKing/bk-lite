# 6.2 版本日志
版本发布时间：2025年11月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 节点管理 | 云区域-首页：支持部署云区域 容器版 、卡片展示Nats Executor、Stargazer组件状态 |
| 节点管理 | 节点管理 云区域-节点：移除配置页面，整合到节点-详情抽屉页 |
| CMDB | 配置采集任务详情，可以查看单条数据的异常日志 写入CMDB的异常 |
| CMDB | 配置采集任务详情新增原始数据，可以看到凭据错误、网络不通导致的采集数据异常的问题 当前生效范围不包括k8s、云平台、VC |
| OpsPilot | Chatflow 新增 Web 对话框，支持 AG-ui 协议接入其他系统，多会话智能管理 |
| 系统管理 | 新增 报错日志管理，快速筛选异常，轻松排查问题 |
| 系统管理 | 新增 密码设置功能，支持强密码策略，提高账号安全 |
| 日志中心 | 支持 SNMP Trap 日志接入，实时告警与数据分析 |
| 监控中心 | 支持 Excel 批量导入采集配置，快速部署监控项 |
| OpsPilot | Chatflow 支持对话历史回溯：完整复原用户对话链路，助你更快理解意图 |
| OpsPilot | 工具管理升级：新增工具集展示，更清晰的工具结构，让配置更高效 |
| 系统管理 | 新增操作日志管理：统一查看权限与配置变更，审计透明度进一步提升 |

### 功能优化

| --- | --- |
