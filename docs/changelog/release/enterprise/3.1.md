# 3.1 版本日志
版本发布时间：2026年5月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 新增用户操作：禁用、启用、解锁（密码输入次数超限后解锁） |
| CMDB | 新增12类中间件自动采集能力 |
| CMDB | 支持将 节点管理 中的主机自动同步至 CMDB 资产 |
| CMDB | 新增 SNMP、IPMI 采集测试工具，便于快速验证采集连通性与配置正确性 |
| CMDB | 自动发现页面新增采集任务概览统计 |
| CMDB | 支持手动上传配置文件 |
| CMDB | 支持查看配置文件不同版本内容，并进行差异对比 |
| 监控中心 | 新增 InfluxDB 监控能力 |
| 监控中心 | 新增 MySQL 监控分析视图 |
| 监控中心 | 告警事件支持点击日期查看对应事件 |
| 监控中心 | Kafka 监控新增 Topic、Group 过滤能力 |
| 监控中心 | 新增华为 OceanStor 存储监控能力 |
| 监控中心 | 告警通知渠道支持多选 |
| 日志中心 | 新增内置日志分析视图，覆盖常见数据库、中间件等 15 类场景 |
| 告警中心 | 新增 Zabbix、SNMP Trap 告警接入通道 |
| 告警中心 | Event 集成新增归属组织定义能力 |
| 告警中心 | 新增告警级别管理能力，支持统一管理事件、告警、事故等级 |
| 告警中心 | 新增「相关告警推荐」能力，可通过相关性规则自动匹配并展示关联告警 |
| 运营分析 | 支持在单值、Top 等视图中展示周期相对变化，帮助快速识别趋势波动 |
| 运营分析 | 新增内置监控中心仪表板，快速掌握监控纳管情况 |
| OpsPilot | 新增记忆能力，支持在 workflow 中读取与写入记忆，帮助沉淀经验、复用上下文 |

| 节点管理 | 修复 Windows 控制器安装异常问题 |
| 节点管理 | 修复 Windows 节点内置 Vector、Winlogbeat 异常问题 |
| CMDB | 优化 K8s 采集安装流程 |
| 监控中心 | 优化指标管理页面支持复制信息 |
| 监控中心 | 修复告警模板指标映射错误问题 |
| 监控中心 | 修复主机节点因缺少 MIB 翻译器导致 SNMP 监控异常的问题 |
| 日志中心 | Syslog、SNMP Trap 上报节点调整为 云区域 IP |

| 告警中心 | 优化告警集成页面，提升配置体验 |
| 运营分析 | 优化内置 CMDB 与告警中心分析视图 |

| OpsPilot | 优化智能体工具调用视觉体验 |
