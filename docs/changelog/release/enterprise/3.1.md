# 3.1 版本日志
版本发布时间：2025年10月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| OpsPilot | Chatflow 新增触发器类型：支持 企业微信、钉钉、微信公众号 等多种触发方式，灵活接入 Chatflow，拓展更多自动化使用场景。 |
| 系统管理 | 新增超级管理员功能：集中化管理权限与配置，提升运维效率。 |
| 系统管理 | 新增虚拟团队功能：支持灵活的团队组织与协作管理。 |
| 节点管理 | 探针管理优化：采集器与控制器拆分管理，结构更清晰、维护更高效。 |
| 节点管理 | 云区域优化：节点与配置管理页面交互升级，前端展示更美观。 |
| OpsPilot | Chatflow 新增一类节点，可以直接使用平台的通知渠道进行消息通知 |
| OpsPilot | lobechat的应用支持记录对话记录和运营统计 |
| 节点管理 | 探针配置安全加固：采集配置文件中的明文密码改为环境变量注入 |
| 节点管理 | 安装过程可视化：控制器安装日志分步骤展示，每一步执行结果清晰可见，错误定位更快捷 |
| 节点管理 | 通讯加密增强：节点与服务之间通讯支持 TLS |
