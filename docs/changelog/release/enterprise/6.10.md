# 6.10 版本日志
版本发布时间：2026年7月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 作业管理 | 作业执行过程中，作业详情页支持流式展示执行进度与结果 |
| CMDB | 新增网络设备配置采集能力，支持统一采集与管理网络配置 |
| CMDB | 新增应用拓扑视图，帮助快速了解应用关系与依赖结构 |
| 监控中心 | 新增 TCP 拨测能力，支持对端口连通性与服务可用性进行探测 |
| 监控中心 | 监控列表支持自定义展示字段，满足不同场景下的查看需求 |
| 监控中心 | 监控接入流程支持连通性测试，提前发现网络、端口或认证配置问题 |
| 运营分析 | 数据源类型进一步拓展，支持 MySQL、PostgreSQL、REST API、Excel |
| OpsPilot | 工作台支持对接企业微信机器人，便于将智能能力接入企业协作场景 |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 监控中心 | 优化监控视图采集断点识别能力，异常数据中断更容易定位 |
| OpsPilot | 对外通知节点支持配置 LLM 模型智能优化通知内容，让消息表达更清晰易懂 |
