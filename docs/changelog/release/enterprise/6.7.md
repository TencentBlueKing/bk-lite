# 6.7 版本日志
版本发布时间：2026年4月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 作业管理 | 上传 Playbook 时支持模板下载 |
| CMDB | 新增主机配置文件采集能力 |
| 监控中心 | 支持自定义 SNMP 采集模板 |
| 监控中心 | 新增 etcd 监控能力 |
| 告警中心 | 新增 K8s Event 接入通道 |
| 运营分析 | 内置 告警中心与CMDB 分析仪表盘 |
| OpsPilot | 工具支持配置多个实例 |
| OpsPilot | 新增 MySQL、Oracle 工具 |
| 平台 | 支持用户修改语言和时区后即时生效，无需重新登录 |
| 系统管理 | 通知渠道配置页面支持测试功能 |
| CMDB | 新增 Elasticsearch、HBase、MongoDB、TiDB、MSSQL 自动发现能力 |
| CMDB | 新增物理服务器 IPMI 数据采集能力 |
| 日志中心 | 新增 K8s 日志采集能力 |
| 运营分析 | 新增公共筛选能力 |
| 系统管理 | 支持将父组织角色继承至子组织 |
| CMDB | 新增资产自动关联能力：支持通过规则自动建立资产间关联关系 |
| CMDB | 数据订阅支持基于模型关系配置触发条件 |
| 运营分析 | 支持仪表盘、拓扑/架构视图、数据源、命名空间的导入与导出 |
| 运营分析 | 新增表格类型组件 |
| 节点管理 | 节点默认开启多类日志采集探针组件 |
| 监控中心 | 开放 监控对象 管理功能 |
| 监控中心 | 支持通过 API PULL / PUSH 模式 自定义上报监控数据 |
| CMDB | 配置采集任务支持自动清理下架资产 |
| CMDB | 模型支持多字段联合唯一性校验 |
| CMDB | 模型配置支持导入与导出 |
| OpsPilot | 工作台支持在线测试 |
| OpsPilot | 新增工具：Agent Browser |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 节点管理 | 重构控制器安装流程，解决因体积过大导致的安装超时问题 |
| OpsPilot | 优化意图分类节点体验，支持脱离智能体独立使用 |
| 监控中心 | 修复 自定义监控上报无法自动创建资产 的问题 |
| 日志中心 | 优化日志搜索体验：支持根据搜索结果自动过滤并展示相关字段 |
