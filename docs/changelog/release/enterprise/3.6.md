# 3.6 版本日志
版本发布时间：2026年3月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 作业管理 | 支持基于 Ansible 的作业执行 |
| 日志中心 | 支持通过左侧字段列表查看 Top 值分布情况 |
| 告警中心 | 新增「缺失检测」策略，用于检测预期应到但未到达的事件 |
| CMDB | 新增 数据订阅 功能，支持在资产属性变化、临近到期时进行通知 |
| CMDB | 主机 Host 采集新增 进程采集 功能 |
| 作业管理 | 平台上线 作业管理 模块，支持对节点管理已纳管的主机执行脚本或上传文件 |
| OpsPilot | 新增内置 Redis、Elasticsearch 运维工具 |
| CMDB | 新增 公共选项库 功能，配置后可在枚举字段选用 |
| CMDB | 枚举字段支持多选 |
| CMDB | Nginx 等中间件采集自动创建与主机的关联关系 |
| 节点管理 | Linux控制器安装兼容低版本SSH |
| 节点管理 | Linux控制器安装兼容无sudo命令环境 |
| OpsPilot | MCP 新增 streamable 传输类型支持 |
| CMDB | 新增 表格 和 标签 类型字段，进一步强化资产属性管理能力 |
| 监控中心 | 支持将告警事件自动同步到 告警中心 |
| MLOps | 提供模型发布后的接入指引 |
| OpsPilot | 新增内置MSSQL工具，增强数据库管理与运维能力 |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| 平台 | 优化用户 Token 到期处理：页面弹出提示窗口，替代直接跳转登录页 |
| 节点管理 | 优化托管组件操作体验，支持查看执行步骤及完成进度 |
| 告警中心 | 优化 UTC 时区处理，提升告警通知时间准确性 |
| OpsPilot | 优化模型添加交互体验，支持同一供应商模型批量管理 |
| 节点管理 | 优化 控制器安装 体验，支持在安装时查看步骤完成情况 |
| 日志中心 | 日志集成 全面升级，提供丰富的场景化日志接入能力 |
| 监控中心 | 重构 搜索 页面，支持灵活搜索指标数据进行组合分析 |
| MLOps | 优化能力发布逻辑，自动寻找并占用可用端口号 |
