# 6.3 版本日志
版本发布时间：2025年12月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| OpsPilot | 工具参数新增支持：密码/数字/Bool类型 |
| 节点管理 | 新增 控制器版本号管理机制，并支持在节点列表中展示 |
| CMDB | 资产页面新增 多条件筛选能力 |
| CMDB | 运营分析内置数据源，支持 告警趋势、日志命中数、单监控指标查询 |
| 监控中心 | 新增 腾讯云资源监控能力 |
| OpsPilot | WorkFlow新增 意图识别节点 |
| 节点管理 | 新增：支持 Windows 节点手动安装控制器 |
| CMDB | 支持通过 SSH 自动采集物理服务器硬件与资源指标 CPU / 内存 / 磁盘等 |
| CMDB | 实例与采集任务可归属多组织 |
| 监控中心 | 各类监控对象的接入能力全面调优 |
| 日志中心 | Docker日志接入能力优化 |
| OpsPilot | 机器人新增图片能力支持 |
| CMDB | 配置采集现已支持查看 全部分类 |
| CMDB | 插件卡片新增 任务概览 展示 |
| CMDB | 采集任务捕获并展示“同步数据”异常日志 |
| CMDB | 采集任务详情新增 概览统计 |
| 监控中心 | 新增 K8s 监控接入 |
| 日志中心 | Syslog 日志接入优化 |
| OpsPilot | Chatflow 应用新增三种触发方式：移动端、嵌入对话、Web |
| OpsPilot | 移动端能力上线，可从移动端入口发起智能对话并查看历史记录 |
| OpsPilot | Web 端接入，支持应用查询与对话交互 |
