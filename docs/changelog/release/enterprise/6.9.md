# 6.9 版本日志
版本发布时间：2026年6月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 系统管理 | 内置系统管理员、安全管理员、审计管理员角色，满足三员管理要求 |
| CMDB | 新增模型布局管理能力，支持分类与模型拖拽排序 |
| CMDB | 配置采集任务支持配置多凭据 |
| 监控中心 | 新增远程主机监控能力，无需安装 Agent 即可远程采集主机监控数据 |
| 监控中心 | 新增内置监控仪表盘，覆盖数据库、中间件等 17 类监控对象 |
| 告警中心 | 相关性规则新增“即时告警”类型，命中事件可立即生成告警 |
| 运营分析 | 仪表盘新增可折叠分组能力，复杂看板可按分组收起与展开 |
| 运营分析 | 分析视图支持全屏查看，提升大屏展示与数据分析体验 |
| 节点管理 | 支持在页面远程卸载手动安装的节点 |
| 作业管理 | 作业详情支持查看 Playbook 模版 |
| CMDB | 新增自定义上报资产数据能力 |
| CMDB | 增强网络拓扑发现能力 |
| 监控中心 | 监控视图列表新增指标配置展示能力 |
| 监控中心 | 新增 WMI 远程监控 Windows 主机能力 |
| 告警中心 | 告警分派策略新增“告警升级”配置能力，未及时响应的告警可自动升级处理 |
| 运营分析 | 仪表盘表格组件支持配置操作列，并新增下钻能力 |
| OpsPilot | 新增 Markdown 文件生成工具，支持智能体生成结构化文档 |
| OpsPilot | 邮件通知节点支持将智能体生成的文件作为附件发送 |
| CMDB | 新增机房机柜视图，支持机柜布局与 U 位利用率展示 |
| CMDB | 新增 IP 地址管理（IPAM）能力，支持网段规划、IP 自动发现扫描与自动对账冲突管理 |
| CMDB | 扩充配置采集对象，新增华为存储、MinIO、InfluxDB 等采集能力 |
| 监控中心 | 新增 117 个 SNMP 监控模板，覆盖多类网络设备品牌与型号 |
| 监控中心 | 新增 HAProxy 监控能力，完善负载均衡场景监控覆盖 |
| 监控中心 | 支持批量配置告警策略模板，提升监控策略配置效率 |
| 日志中心 | 强化日志告警策略配置体验，支持告警预览，并支持在告警名称中引用变量 |
| 告警中心 | 新增告警自动化处置能力，可按规则自动触发作业并记录执行结果 |
| 告警中心 | 新增告警丰富（Enrichment）能力，支持基于 CMDB 资源等信息自动补充告警上下文 |
| 运营分析 | 内置告警分析大屏，帮助快速掌握告警趋势与处置情况 |
| 运营分析 | 新增网络拓扑组件，支持在分析视图中呈现网络关系 |
| 运营分析 | 分析视图支持接入系统管理操作日志 |
| OpsPilot | 工作台新增 NATS 触发节点，支持由告警中心触发 AI 工作流 |

### 功能优化

| 模块 | 功能优化 |
| --- | --- |
| OpsPilot | 优化智能体人机协同决策交互，提升选项确认与流程推进体验 |
| 监控中心 | 优化监控视图展示逻辑，仅呈现已采集的模版指标 |
| 日志中心 | 合并 HTTP 流量与网络流量采集模版，并开放监听网卡配置项 |
| 作业管理 | 优化首页展示内容，提升作业入口信息呈现效率 |
