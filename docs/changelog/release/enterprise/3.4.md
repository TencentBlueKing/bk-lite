# 3.4 版本日志
版本发布时间：2026年1月

### 功能新增

| 模块 | 新增功能 |
| --- | --- |
| 节点管理 | 编辑窗口支持全屏操作 |
| CMDB | 新增搜索条件收藏功能 |
| CMDB | 网络设备与网络拓扑采集任务合并，按需选择是否采集关系数据 |
| 告警中心 | 引入智能降噪能力：支持对 Event 事件进行多维度自动识别与合并 |
| OpsPilot | 知识库支持上传图片文件 |
| OpsPilot | 支持查看文档处理失败原因 |
| OpsPilot | 智能体页面补充工具能力说明 |
| CMDB | 新增 Postgres 与 Docker 资产采集能力 |
| CMDB | 支持直接在模型分组下新增模型，优化交互体验 |
| OpsPilot | 内置 PostgresDB 巡检助手、K8s巡检助手、Web应用巡检助手，调整配置即可使用 |
| CMDB | 新增 模型复制功能 |
| CMDB | 全文检索支持 大小写模糊匹配，支持枚举、用户、组织等显示名与存储数据映射的字段 |
| CMDB | 实例拓扑支持 持续向下一级展开 |
| OpsPilot | 定时任务执行时间调整为 优先读取用户设置的时区 |
| OpsPilot | 新增 PostgresDB 巡检助手、K8s巡检助手 ReAct 场景验证 |
| 节点管理 | 支持通过 密钥凭据 远程安装 Linux 节点 |
| CMDB | 模型管理新增 字段分组能力 |
| CMDB | 配置采集新增 Windows Server 支持 |
| 监控中心 | 新增 指标单位自动转换 |
| OpsPilot | 新增 BrowserUse 工具，智能体可操作浏览器完成自动化巡检任务 |
| OpsPilot | 增强知识库对 文档图片内容的识别能力 |
