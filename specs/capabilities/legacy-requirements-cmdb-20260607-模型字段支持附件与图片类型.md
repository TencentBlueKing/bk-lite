# 模型字段支持附件与图片类型

> Migrated from `spec/requirements/CMDB/20260607.模型字段支持附件与图片类型.md` as legacy capability evidence.

日期：2026-06-07

## 背景

CMDB 资产实例存在文件型信息需求：合同管理需要在实例上挂载合同文档，设备管理需要记录设备外观等图片。现有字段类型均为文本/枚举/结构化数据，无法承载文件本体。

为此，CMDB 模型字段新增「附件」与「图片」两种通用字段类型，作为与现有字段类型同级、可被任意模型自助配置的字段类型，而非定制业务模块。合同、设备外观仅为示例场景。

## 需求描述

### 1）字段定义

- 模型字段类型新增「附件」与「图片」两类。
- 两类字段均支持在一个字段下挂载多个文件。
- 字段配置仅设置字段名称、所属分组、字段类型，不暴露文件相关约束旋钮，约束由系统统一兜底。
- 附件/图片字段一律可选，不可设为必填；字段类型创建后不可切换。

### 2）实例维护

- 实例新增/编辑时，附件/图片字段支持逐个上传与删除文件，超出系统约束时给出明确提示。
- 详情页：附件展示文件名列表并提供下载；图片展示缩略图并支持放大预览。
- 下载权限跟随实例的读权限，不单独设置文件级权限。

### 3）列表与检索

- 列表中附件/图片字段仅展示摘要（如文件数量或缩略图），不可排序、不可筛选、不参与搜索。

### 4）系统级约束（统一兜底）

| 约束 | 默认值 |
|---|---|
| 单字段最大文件数 | 附件 5 / 图片 5 |
| 单文件大小上限 | 附件 50MB / 图片 10MB |
| 附件允许类型 | 常见办公与压缩类（pdf/doc/xls/ppt/txt/zip 等） |
| 图片允许类型 | jpg/png/gif/webp/bmp |
| 必填 | 永远可选 |

### 5）与现有规则的边界

- 不可加入联合唯一规则，不参与自动关联规则匹配。
- 上传/替换/删除文件视为实例编辑，生成变更记录，快照体现文件名列表变化。

## 关键技术架构选择

- 文件本体存于对象存储，实例字段仅保存文件元数据与对象引用，沿用配置文件已有的对象存储方案。
- 图片复用附件的存储与上传下载基建，仅在允许类型与展示渲染上不同。
- 文件回收（删除实例/字段、移除文件后）由异步任务处理，与对象存储保持最终一致。

## 适用范围

- 模型字段定义与编辑
- 实例录入与修改
- 实例详情展示
- 实例列表展示（仅摘要）

## 非目标（本期不包含）

- 批量导入、批量导出：附件/图片字段不进入 Excel 导入导出。
- 搜索/筛选：附件/图片字段不参与实例检索与筛选。
- 字段级约束自定义：管理员不可设置文件数、大小、类型、必填等约束。
- 文件级权限：不提供独立于实例读权限的下载权限控制。
- 在线预览编辑（图片放大预览除外）、文件版本管理。

## 验收标准

- 模型字段可新增附件与图片类型，字段配置仅含名称、分组、类型。
- 实例可多文件上传与删除并保存；附件可下载，图片可缩略图查看与放大预览。
- 下载权限跟随实例读权限：能查看实例即可下载其附件，反之不可。
- 系统级约束生效：超过文件数、大小上限或非允许类型时上传被拦截并提示。
- 附件/图片字段不出现在搜索、筛选、Excel 导入导出中。
- 附件/图片字段无法加入联合唯一规则，无法参与自动关联匹配。
- 上传/替换/删除文件生成变更记录，快照体现文件名列表变化。
- 删除实例/字段或移除文件后，对应对象存储文件被异步回收。
