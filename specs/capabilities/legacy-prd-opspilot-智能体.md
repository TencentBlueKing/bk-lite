# OpsPilot · 智能体

> Migrated from `spec/prd/OpsPilot/智能体.md` as legacy capability evidence.

## 1. 背景与目标

智能体定义大模型的行为：模型选择、提示词、工具与知识检索（RAG）配置，是机器人对话能力的核心单元。

## 2. 范围定义

- 基础配置：模型、提示词、温度、对话窗口。
- 工具绑定：函数调用工具。
- 知识检索（RAG）：知识库绑定与检索策略。
- 高级能力：知识路由、查询改写、计划执行、追问建议等。

## 3. 关键能力

- 基础配置：从模型库选择 LLM，配置系统提示词、温度、保留的历史对话条数。
- 工具绑定：绑定函数调用工具（内置数据库/缓存/HTTP 等或自定义）。
- 知识检索（RAG）：
  - 开关 RAG，绑定一个或多个知识库。
  - 按知识库设置相关性分数阈值。
  - 严格模式：仅在检索到知识时回答。
  - 在回答中展示知识来源（引用）。
- 高级能力：知识库自动路由、查询改写、计划-执行多步推理、追问建议。
- 支持内置与可导入的智能体模板。

## 4. 关键规则

- 智能体按组织隔离。
- 各知识库可分别设置检索分数阈值，实现按库精度控制。
- 对话历史窗口大小可配置。

## 5. 关键技术架构选择

- 基于 LangChain/LangGraph 编排，支持工具调用与多步推理。
- RAG 管线：嵌入 → 向量检索 → 可选重排 → 提示词增强。

## 6. 验收标准

- 用户可配置智能体的模型、提示词、工具与 RAG 策略。
- RAG 可按知识库阈值过滤检索结果并按需展示来源。
- 高级能力（知识路由、查询改写、计划执行、追问建议）可按配置生效。
