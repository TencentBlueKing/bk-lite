# 监控视图个人列配置

## 目标

监控视图列表允许当前登录用户选择展示字段并调整顺序。配置按监控对象保存，只影响本人，刷新或重新登录后仍生效。

## 行为契约

- 列表右上角提供可访问的字段设置按钮。
- 设置窗口展示当前对象的基础字段和指标字段，支持搜索、勾选及拖拽排序。
- 操作列不可配置，始终固定在表格最右侧。
- 用户未保存配置时展示对象当前全部默认字段。
- 保存配置后，列表只展示仍然存在的已选字段，并按保存顺序排列。
- 对象字段发生变化时忽略已失效字段；如果保存字段全部失效，回退当前默认字段。
- 配置以用户和监控对象为唯一范围，不因标题翻译或系统默认排序变化而失效。
- 个人配置读取失败不得阻断监控实例列表展示；保存失败不得覆盖当前已生效配置。

## 接缝

- `GET/PUT /monitor/api/monitor_object/{id}/view_column_preference/`
- 前端 `resolveViewColumns(availableColumns, savedFieldKeys)` 纯函数

## 验证

- 后端接口覆盖保存、读取、输入校验及用户隔离。
- 展示字段接口覆盖稳定列键不受标题、语言和默认排序影响。
- 前端纯逻辑覆盖默认列、个人顺序、失效字段过滤和全失效回退。
